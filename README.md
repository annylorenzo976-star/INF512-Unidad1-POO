# INF512-Unidad1-POO
El proyecto incluye una clase llamada 'persona ' que representa a una  persona con:
- *Atributos*:
  - Nombre: almacena el nombre de la persona.
  - Edad: almacena la edad de la persona.

- *Métodos*:
  - Saludar(): muestra un mensaje de saludo con el nombre y la edad.
  - Cumpleanos(): aumenta la edad en 1 y muestra un mensaje de cumpleaños.
  - Despedirse(): muestra un mensaje de despedida.

El archivo Program.cs crea un objeto Persona y prueba todos sus métodos.
